# UiPath Maestro BPMN Integration Guide

To ensure this project qualifies for the **UiPath Maestro BPMN Track**, you need to demonstrate in your submission video that **UiPath Maestro** is the top-level orchestrator that governs the entire multi-agent LangGraph workflow.

Since we built the LangGraph engine in Python, we will use a **UiPath Maestro BPMN Process** to trigger the API, wait for the AI to finish, and handle the final enterprise routing.

## The BPMN Architecture

You need to build a BPMN workflow in UiPath Studio Web that visually resembles this flow:

### 1. Start Event (Alert Trigger)
*   **Trigger:** The process begins when a new AML alert is generated (this could be a webhook from a core banking system or a file drop).
*   **Action:** Triggers the Maestro BPMN.

### 2. Service Task (Call External AI Agents)
*   **Node:** "Invoke AI Agents (LangGraph)"
*   **Action:** This node makes an HTTP Request to our Python FastAPI backend (`http://your-server-ip/api/investigate/stream`).
*   **Concept:** This perfectly demonstrates the hackathon requirement: *"blend UiPath-native agents with external agents."* Maestro delegates the heavy unstructured "thinking" to the specialized LangGraph agents.

### 3. Exclusive Gateway (Risk Decision)
*   **Node:** "Evaluate Risk Score"
*   **Condition:** Maestro evaluates the `final_risk_score` returned by the LangGraph API.
    *   **Path A (High Risk > 70):** Route to Human-in-the-Loop.
    *   **Path B (Low Risk <= 70):** Auto-close the alert (Straight Through Processing).

### 4. Service Task (Data Persistence)
*   **Node:** "Push Evidence to Data Service"
*   **Action:** Maestro uses native UiPath Data Service activities to permanently store the structured FinCEN Form 111 JSON payload generated by LangGraph.

### 5. User Task (Action Center HITL)
*   **Node:** "Review SAR"
*   **Action:** If the risk was high, Maestro natively suspends the job and creates an Action Center task for the BSA Officer. Maestro enforces the governance rule that an AI cannot file a SAR without a human signature.

### 6. End Event
*   **Node:** "Alert Resolved"
